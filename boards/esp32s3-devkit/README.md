# ESP32-S3 DevKit (and clones)

Based on [Espressif ESP32-S3-DevKitC-1][devkitc]. Espressif version uses micro USB. v1.0 uses GPIO48 for RGB led, while v1.1 uses GPIO38. [Espressif ESP32-S3-DevKitM-1][devkitm] is functionallty the same as DevKitC-1 v1.0.

Clones usually based on DevKitC-1 v1.0, use USB-C, have different layouts,
and may swap some components for functional equivalents (e.g. LDO, USB bridge).
